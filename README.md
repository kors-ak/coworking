# Coworking Landing Page

_Учебный проект по frontend-разработке_

Адаптивный лендинг для коворкинг-пространства, разработанный с использованием HTML5 и SCSS.

## О проекте

Современный адаптивный лендинг, представляющий услуги коворкинга, его преимущества и форму обратной связи. Проект демонстрирует использование семантической HTML-разметки, методологии БЭМ, архитектуры SCSS и принципов адаптивной верстки.

- Семантическая HTML5-разметка
- Адаптивный интерфейс для компьютеров, планшетов и мобильных устройств
- Использование SCSS с переменными и вложенностью
- Именование классов по методологии БЭМ
- Плавная прокрутка страницы
- Верстка на Flexbox и CSS Grid
- Форма обратной связи
- Оптимизированные изображения в формате WebP

## Используемые технологии

- HTML5
- SCSS (Sass)
- CSS3
- Flexbox
- CSS Grid
- БЭМ (BEM)

## Запуск проекта

1. Клонируйте репозиторий:

```bash
git clone https://github.com/kors-ak/coworking.git
```

2. Откройте файл `index.html` в браузере.

Для удобной локальной разработки рекомендуется использовать расширение **Live Server** для Visual Studio Code.
